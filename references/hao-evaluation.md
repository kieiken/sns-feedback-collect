# Hao Evaluation Notes

Use these when recommending whether SNS投稿90日計画 needs adjustment after collecting reactions.

## Timing

- Do not make a final success/failure call before 48-72h plateau.
- 1-6h: report only early reach and trajectory.
- 6-24h: report current trend, not final judgment.
- 24-48h: watch for second push; avoid overfitting.

## Signal Priority

Hao's evaluation priority:

1. Private shares / DM shares if visible
2. Saves
3. Meaningful comments, especially 5+ characters
4. Public shares
5. Ordinary comments
6. Dwell / read-more behavior if visible
7. Reactions/likes

Reactions alone are weak. A post with low likes but high saves, shares, meaningful comments, link clicks, or downstream conversion can still be successful.

## Useful Derived Checks

- Interaction rate: `interactions / reach`
- Strong reactions: `shares + saves + meaningful_comments`
- Strong reaction rate: `strong_reactions / reach`
- For Threads, use visible views when unique reach is unavailable.

## Adjustment Heuristics

- If a post reaches the intended country/audience but gets no comments/saves/shares, keep the topic direction and improve the interaction design.
- If follower-outside percentage is under 15% for three posts in a row, the account may be locked in the existing follower circle. Use broader hooks, concrete demos, or clearer stance.
- If a post is under 24h old and follower-outside reach is weak, do not rewrite the plan yet; schedule a 48-72h recheck.
- If comments are mostly the user's own link/口火 comments, count them as operations, not audience engagement.
- If the content is abstract stance-only and gets no saves/shares, add a concrete scene, example, demo, checklist, or "save-worthy" framing.
- For 繁中 posts, if Taiwan share is high but comments are zero, keep the Taiwan route and add a five-second answerable question.
- For pure F19 stance posts, preserve stance clarity but make the first sentence more concrete when early non-follower reach is weak.

## Recommendation Shape

Give a small decision:

- `No major change`: direction is sound; continue and recheck at plateau.
- `Micro-adjust`: keep calendar/positioning, change hook, mouth-of-comment, save trigger, or question design.
- `Route change`: repeated low non-follower reach after plateau; switch formula/archetype.

Avoid large rewrites from small samples.
